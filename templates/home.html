Design and Development of a Hybrid Deep Learning Model for Student Performance Prediction in MOOCs: Integrating Optimized Feature Selection

Project advisor: Ts.Dr.Chee Ken Nee

Project Overview

Design and Development of a Hybrid Deep Learning Model for Student Performance Prediction in MOOCs: Integrating Optimized Feature Selection leverages advanced technology to forecast student performance in online courses. By analyzing key data points, we predict outcomes such as passing, failing, withdrawing, or achieving distinction, enabling timely interventions to boost success rates.

This initiative aims to empower educators and institutions with insights to enhance student retention and academic achievement.

Technical Approach

Our approach utilizes a CNN+LSTM model, a powerful combination for pattern recognition and time-series analysis. Here’s a breakdown:

  1. Data Collection: We gather 12 features, including grades and online engagement metrics.
  2. Pattern Detection (CNN): Identifies relationships within static data, like quiz performance.
  3. Temporal Analysis (LSTM): Tracks changes over time, such as increasing participation.
  4. Prediction: Combines insights to deliver accurate outcome forecasts.

This hybrid model outperforms traditional methods by deeply learning from complex datasets.

Features and Data

We base our predictions on 12 features categorized into three areas:

  • Academic Background: Course attempts, credits studied.
  • Engagement Metrics: Forum posts, resource views, quiz attempts, tool usage.
  • Demographics: Education level, income indicators, age, disability status.

These features are refined and standardized to optimize model performance.

Project Goals

  • Develop a highly accurate prediction tool.
  • Enable early identification of at-risk students.
  • Identify key factors influencing success.
  • Create an intuitive web interface for exploring predictions.

Success here could transform online education outcomes.
