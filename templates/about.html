About the Student Outcome Prediction System

Project Overview

The Design and Development of a Hybrid Deep Learning Model for Student Performance Prediction in MOOCs: Integrating Optimized Feature Selection is an innovative project aimed at predicting student performance in Massive Open Online Courses (MOOCs). By leveraging a hybrid CNN+LSTM model, the system analyzes academic records, online engagement metrics, and demographic data to forecast outcomes (Pass, Fail, Withdrawn, Distinction). This enables educators to identify at-risk students early and provide targeted interventions to enhance success rates.

Our goal is to transform online education by combining advanced machine learning with optimized feature selection, making learning more effective and accessible worldwide.

Objectives

  • ◆ Develop a highly accurate model to identify students at risk, enabling timely interventions.
  • ◆ Create an intuitive web interface for educators to input data and receive real-time predictions.
  • ◆ Visualize key data insights through charts to support data-driven decisions.
  • ◆ Contribute to educational research by analyzing factors influencing MOOC outcomes.

Methodology

The system employs a hybrid CNN+LSTM model, integrating:

  1. 1 Convolutional Neural Network (CNN): Detects patterns in student data, such as correlations between quiz scores and forum activity.
  2. 2 Long Short-Term Memory (LSTM): Captures temporal trends, like changes in engagement over time.
  3. 3 Data Processing: Normalizes 12 features using StandardScaler for effective learning.
  4. 4 Prediction: Outputs one of four outcomes (Pass, Fail, Withdrawn, Distinction) with high accuracy.

Trained on a curated MOOC dataset, the model leverages optimized feature selection to outperform traditional methods.

Features and Data

The model uses 12 carefully selected features, categorized as:

  • ◆ Academic Background: Previous course attempts, studied credits.
  • ◆ Engagement Metrics: Forum participation, content views, quiz attempts, resource usage.
  • ◆ Demographic Details: Education level, socio-economic status, age, disability.

These features are optimized to maximize prediction accuracy and reliability.
